Задание «Создание пользовательских интерфейсов Master-Detail»

Вариант 2. Средний
Разработать WPF-приложение с использованием паттерна MVVM.
В главном окне (Master) отобразить список элементов с несколькими колонками (например, список сотрудников с ФИО, должностью и отделом).
При выборе элемента открывать детальное представление (Detail) в отдельном окне с возможностью редактирования всех данных.
Реализовать функциональность поиска по списку с возможностью фильтрации элементов.
Добавить валидацию данных, запрещающую ввод некорректных значений (например, возраст — только числа).
Обеспечить возможность загрузки данных из внешнего JSON-файла и сохранения изменений обратно в файл.
Использовать команды и привязки для обработки пользовательских действий (например, команды "Добавить", "Удалить", "Сохранить").
Обеспечить обработку ошибок и исключений при работе с файлами и некорректными данными.
В отчете представить описание архитектуры решения, используемых технологий и демонстрацию работы приложения.
